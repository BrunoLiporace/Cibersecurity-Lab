# Modulo 01: Database Fundamentals & SQL CLI Operations

## Objective
This repository contains the practical inplementation and automation scripts for the Database Fundamentals module. The core objective is to master relational database design, data definition, and data manipulation strictly through the Linux Command Line Interface (CLI), replicating an enterprise production enviroment.

All exercises and structural designs are based on standard relational database constraints, strict data typing, and advanced filtering tchniques.

---

## Environment & Prerequisites 
* **Operating System:** Debian GNU/Linux 13 (Trixie)
* **Database Engine:** MariaDB Server 11.x (MySQL compatible)
* **Access Method:** Remote Terminal via SSH (No GUI / No Workbench)
* **Required Utilities:** `mariadb-client`, `bash`, `micro`/`vim`
