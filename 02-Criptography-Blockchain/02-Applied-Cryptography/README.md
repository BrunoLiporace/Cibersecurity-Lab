# Applied Cryptography & Analysis

This module focuses on the practical implementation of cryptographic primitives and the analysis of their vulnerabilities, based on modern standards and historical cryptanalysis techniques.

## Practical Objectives
- **Symmetric Ciphers:** Implementing AES-GCM and understanding block cipher padding (PKCS#7).
- **Asymmetric Systems:** Key exchange protocols (Diffie-Hellman) and Hybrid Encryption (HPKE).
- **Cryptanalysis:** Statistical analysis of ciphertexts and Side-Channel attack theory.

## Tools & Requirements
- Python 3.x
- `pycryptodome` or `cryptography` libraries.
